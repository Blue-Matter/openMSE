openMSE is designed to be flexible and extensible. Users create fully customized operating models, or import operating models from stock assessments.

openMSE includes a library of existing operating models that can be used or modified, as well as a range of example OM components for quickly building operating models.

The OMs in the OM library are available in the MSEextra package:

# download MSEextra from Github
MSEextra()
