Real fishery data can be added to the OM with OM@cpars$Data <- Data.

When a real Data object is detected in the OM, the observation model is conditioned on this data.

You can learn more about this in the Conditioning OMs with Data section.

In the following sub-sections we demonstrate how to condition the observation model with catch, index, composition, and life-history data.
